# Teslo API

1. Clone repo
2. Up DB
```yarn install ```
3. Clone ```.env.example``` and rename ```.env```
4. Change values in environment variables
5. Up DB
```docker-compose up -d ```

6. Execute SEED
```
https://localhost:3000/api/seed
```

7. Run app: ```yarn start:dev```
